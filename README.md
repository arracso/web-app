This project is set using npm from [node.js](https://nodejs.org/es/).
You can use any version of node.js and npm from version 8.11.2.

## Installation
```shell
cd web_app
npm install
```

## Run - Development
```shell
npm start
```

## Build - Production
```shell
npm run build
```

## Build docs
```shell
npm run buildDocs
```
